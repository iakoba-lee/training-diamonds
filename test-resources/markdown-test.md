# 💎 Markdown Master Test File

Welcome to the **Ultimate Markdown Renderer Test**! Use this content to see how your app handles various styles.

## 1. Text Formatting
*   *Italic text* using asterisks or _underscores_.
*   **Bold text** using double asterisks or __double underscores__.
*   ***Bold and Italic*** combining both.
*   ~~Strikethrough text~~ using double tildes.
*   `Inline code` using single backticks.
